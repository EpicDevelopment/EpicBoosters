
# Epic Boosters

Basic Boosters plugin for Minecraft version 1.20.4


## Usage/Examples

```
/grantbooster <player> <drops/experience/damage> <multiplier> <time> [<amount>]
/boosters
```


## Dependencies


 - [Spigot](https://getbukkit.org/get/272245e4f948b0a66b0b4c34dfa27c49)
